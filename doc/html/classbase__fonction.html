Algorithme d'exécution selon un parametre donné dans le cas : ligne, itération sur chaque ligne.

Paramètres
compilLe compilateur utilisé.
textes_inLe texte source en entrée.
textes_outLe texte de sortie généré.
callbackLa fonction à appeler en fin d'algorithme.
Remarques
L'algorithme itère sur les lignes. Seule le premier mot de chaque ligne est considéré.
Les données courantes sont ainsi :
  • la ligne courante
  • un itérateur sur la ligne, i.e. un itérateur sur les mots de la ligne.
